Backend Connection Guide
1

Verify Backend Host Coordinates

The frontend executes predictive API queries to: http://127.0.0.1:8000/predict. Verify your FastAPI microservice binds to that target.
2

Start FastAPI with Reload Mode

Activate your localized prediction pipeline from your repository environment terminal:
uvicorn main:app --host 127.0.0.1 --port 8000 --reload
3

Ensure CORS Policy is Active

To allow browser fetch requests from other origins, activate CORSMiddleware in your FastAPI setup:
from fastapi import FastAPI from fastapi.middleware.cors import CORSMiddleware app = FastAPI() app.add_middleware( CORSMiddleware, allow_origins=["*"], # Allow your frontend app to call predictions allow_credentials=True, allow_methods=["*"], allow_headers=["*"], ) @app.post("/predict") async def predict(file: UploadFile = File(...)): # Your YOLO model inference here return [{"class": "person", "confidence": 0.94}]
